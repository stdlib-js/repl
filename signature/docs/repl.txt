
{{alias}}( alias )
    Returns the signature(s) associated with a provided alias.

    Parameters
    ----------
    alias: string
        Alias.

    Returns
    -------
    out: Array<string>|null
        Signature(s).

    Examples
    --------
    > var out = {{alias}}( '{{alias:@stdlib/math/base/special/sin}}' )
